
# Bulgarian Trainer

A static web app for learning Bulgarian up to B1 with a practical focus on:
- daily vocabulary
- work and mechanic language
- verbs
- listening
- speaking
- practice quizzes
- evaluation tests
- progress tracking

## Run locally
Use any local web server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Upload the files in this folder.
3. In GitHub, go to Settings → Pages.
4. Select the branch and root folder.
5. Save.

The app will work best in Chrome for the audio and speaking features.
